### ✅ Engineering Standards

- **Modular Microservices:** All backend systems designed with event-driven decomposition. Services communicate via RabbitMQ message queues — decoupled, independently deployable, independently scalable.
- **Test Coverage as a First-Class Metric:** Minimum 60% coverage on all active projects. Unit, integration, and E2E coverage via Jest and Playwright. No deployment without a green pipeline.
- **CI/CD on Every Repo:** Automated build, test, and deploy pipelines configured across staging and production environments. Rollbacks are one command.
- **TypeScript Everywhere:** Full type safety from API schema (Zod/Pydantic) to frontend state. Interfaces enforced at the boundary, not assumed in the middle.
- **Performance is a Product Requirement:** Core Web Vitals tracked as KPIs. Bundle sizes are budgeted, not discovered post-deploy.
